# TUTACjs
A JavaScript implementation of the TUTor Automatic Computer (TUTAC), a theoretical computer designed to aid understanding of computers and computer programming techniques. The design of TUTAC was detailed in the 1964 book "Computer Programming Techniques" by Theodore G. Scott, originally published in the US by U.S. Industries, Inc. and in Great Britain by Macdonald & Co. (Publishers) London.
